### 3. **Pipeline Architecture**
   ```
   User Prompt
       ↓
   [Phase-1: main.py]
   - Generates script from prompt
   - Creates character images
   - Outputs: phase1_inputs/scene_manifest.json + character images
       ↓
   [Phase-2: parser.py]  
   - Reads manifest JSON
   - Generates character voices
   - Creates videos with face swaps
   - Outputs: output/raw_scenes/*.mp4
   ```

### 4. **New Unified Pipeline** ✓
   - **File**: `pipeline.py`
   - **Features**:
     - `run_full_pipeline()` - Execute phase-1 → phase-2 in sequence
     - `--skip-phase1` - Use existing manifest (skip image generation)
     - `--resume-phase2` - Resume from checkpoints (continue interrupted video generation)
     - `--mode manual|auto` - Interactive or autonomous execution
   
   **Usage**:
   ```bash
   # Full pipeline with default prompt
   python pipeline.py
   
   # Custom prompt
   python pipeline.py "Your story prompt here" --mode auto
   
   # Skip phase-1, use existing manifest
   python pipeline.py --skip-phase1 --mode auto
   
   # Resume interrupted phase-2
   python pipeline.py --skip-phase1 --resume-phase2
   ```
